PowerShell — расширяемое средство автоматизации от Microsoft 
с открытым исходным кодом, состоящее из оболочки с интерфейсом 
командной строки и сопутствующего языка сценариев. 

$PSVersionTable - переменная, в которой хранится информация о версии
И другая техническая информация о конфигурации

Оболочка для взаимодействия с операционной системой Windows
Проще говоря PowerShell является своего рода интерфейсом 
командной строки для взаимодействия с Windows

Get-ADUser -Identity pvntheraxxx - информация о пользователе pvntheraxxx

powershell -command "get-process -name powershell" 
Можно обращаться к powershell из cmd 

Powershell предлагает свой встроенный скриптовый язык программирования
для автоматизации бизнес-процессов и администрирования 
операционной системы.

Get-ExecutionPolicy -List 
Эта команда выводит информацию о политике безопасности Windows,
Разрешено ли выполнение скриптов и прочее.

Set-ExecutionPolicy unrestricted
