Команды Windows PowerShell

Основные типы команд включают в себя:
1. Псевдонимы (Alias). Используются для сокращения длинных команд.
2. Приложения (Applications). Программы.
3. Командлеты (Cmdlets). Это нативные инструменты PowerShell
4. Функции (Functions). Функции (подпрограммы) позволяют развивать возможности ПО

Get-Command - выводит все имеющиеся командлеты в системе Windows

(Get-Command).count - выводит число-количество доступных команд
(Get-Command -All).count

Get-Command | Select-Object -First 10 - выводит описание первых десяти команд

Для работы с очень сложными и длинными командлетами
Нужно использовать псевдонимы. Это упрощает работу.

Но псевдонимы хранятся только до конца сессии PowerShell.
Это работает не на постоянной основе.
